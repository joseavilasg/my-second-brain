# Installing NVM

## Installing NVM on Ubuntu
```bash copy
sudo apt install curl
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.profile   
```

## Installing Node using NVM
```bash copy
sudo apt install curl
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
```

## Working with NVM
##### List installed versions
```bash copy
nvm ls
```

##### Available node.js version for the installation
```bash copy
nvm ls-remote
```

##### Select a node version
```bash copy
nvm use 12.18.3
```

##### Find default node version
```bash copy
nvm run default --version
```

##### Run node script with other node version
```bash copy
nvm exec 12.18.3 server.js
```
