# Chrome Dark Mode
##### Create a new .desktop file using Chrome's default
```bash copy
cp /usr/share/applications/google-chrome.desktop ~/.local/share/applications/google-chrome.desktop
```
##### Edit the .desktop file that was just created.
```bash copy
gedit ~/.local/share/applications/google-chrome.desktop
```
##### Search for Exec and append --enable-features=WebUIDarkMode --force-dark-mode to the end of the line each time it appears (about 3 times)
```bash copy
# Before
Exec=/usr/bin/google-chrome-stable %U
```
```bash copy
# After
Exec=/usr/bin/google-chrome-stable %U --enable-features=WebUIDarkMode --force-dark-mode
```
