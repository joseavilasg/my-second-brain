# Enable Asus Laptop Battery Health Charging
## Edit file
##### Go to /sys/class/power_supply/ and check if we have BAT1 or BAT0 folder.

```bash copy
cd /sys/class/power_supply/
```

##### Then:
```bash copy
sudo nano /sys/class/power_supply/BAT1/charge_control_end_threshold
```
##### Or:
```bash copy
echo 60 | sudo tee /sys/class/power_supply/BAT1/charge_control_end_threshold
```
The number represents the maximum percentage of battery charge

## Create crontab job
##### We have to create a crontab job, in order to preserve the configuration
```bash copy
sudo nano /etc/crontab
```
##### We add this:
```bash copy
# Battery Charge Threshold by OS Busters
@reboot root echo 60 > /sys/class/power_supply/BAT1/charge_control_end_threshold
#
```
