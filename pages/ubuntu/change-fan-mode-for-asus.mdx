# Change fan mode for ASUS
##### Open file with nano and change number, when:

##### Where:
* 2: Silent
* 0: Balance
* 1: Turbo

```bash copy
sudo nano /sys/devices/platform/asus-nb-wmi/fan_boost_mode
```
