# Numpad Issue
##### Just execute this command
```bash
setxkbmap -option 'numpad:microsoft'
```
##### Then add it to startup applications (Aplicaciones de inicio)
```bash
Name: Make Shift+NumPad work like MS Windows
Description: whatsoever
Command:setxkbmap -option 'numpad:microsoft'
```
Reference [here](https://askubuntu.com/questions/44921/how-to-change-shift-keypad-key-behavior)
